Import PDF Bộ thẻ/Thẻ

Hướng dẫn cấu trúc file PDF:
  • Mỗi bộ thẻ bắt đầu bằng dòng: ##Tiêu đề;chủ đề;miêu tả
  • Các thẻ thuộc bộ thẻ đó nằm ngay sau, mỗi thẻ một dòng: thuật ngữ;định nghĩa
  • Khi là bộ thẻ mới bắt đầu bằng ##, sẽ tạo bộ thẻ mới.
Ví dụ: 
##Toán học;Đại số;Các công thức cơ bản
Phương trình bậc nhất;ax + b = 0
Hàm số bậc hai;y = ax^2 + bx + c
##Tiếng Anh;Từ vựng;Từ vựng cơ bản
Apple;Quả táo
Book;Quyển sách
Cat;Con mèo
